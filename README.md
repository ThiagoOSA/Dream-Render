Para carregar uma cena pressione no botão superior direito e selecione um arquivo .JSON.
Lembre-se que o arquivo .JSON escolhido precisa estar estruturado da mesma forma que o arquivo de cena padrão.
O arquivo de cena padrão pode ser encontrado em /Program/scene_01.json.

To load a scene press the load button on the top right corner and select the .JSON file.
Remember that the .JSON file needs to be structured the same as the default scene file.
The default scene file can be found at /Program/scene_01.json.

W - Para mover para frente.
S - Para mover para trás.
A - Para mover para esquerda.
D - Para mover para direita.
Q - Para rotacionar para esquerda no eixo Y.
E - Para rotacionar para direita no eixo Y.
R - Para rotacionar para cima no eixo X.
F - Para rotacionar para baixo no eixo X.
Z - Para rotacionar para esquerda lateralmente no eixo Z.
F - Para rotacionar para direita lateralmente no eixo Z.

W - To move forward.
S - To move backwards.
A - To move to the left.
D - To move to the right.
Q - To rotate left on the Y axis.
E - To rotate right on the Y axis.
R - To rotate up on the X axis.
F - To rotate down on the X axis.
Z - To rotate left sideway on the Z axis.
F - To rotate right sideways on the Z axis.
